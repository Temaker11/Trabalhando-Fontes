# Trabalhando com fontes

Tipografia transmite mensagem
  - negrito
  - tamanho
  - estilo 

## Basic font properties
  * font-family
  * font- weight
  * font-style
  * font-size

  ## font family
  * Tipo de fonte de um elemento
  * Lista de fontes e ordem de propriedade
  * inclui *fallback* font

  ```css
  p {
    font-family: "Times New Roman", times, serif;
  }
  ```
    - serif
    - sans

    ## Font Weight
    * Peso da fonte

    ```css
    p {
      font-weight: bold;
    }
    ```

    ## Font-Style
    * O estilo da fonte 
    ```css
     span{
      font-style: italic;
     }
     ```
    ## Font Size 
    * o tamanho da fonte
    ```css
    p {
      font-size: 18px;
    }
    ```
